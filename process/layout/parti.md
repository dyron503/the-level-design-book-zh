# 图示

## 图示

建筑学中，**图示**意为项目的核心想法/概念。建筑的图示简化到用一句话就能描述它的空间结构。

关卡设计和建筑设计有一点很类似：核心想法有助于锚定剩余设计。进行这些工作时，定义核心想法都有益处。将自己的关卡简化成草图或图示，会利于合作者（乃至玩家）理解你的意图。
